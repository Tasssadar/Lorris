Jak kompilovat

Budete potřebovat QtFramework >= 4.7 (4.6 by taky mohl fungovat, ale není to vyzkoušené).
Zkompilované binárky by měly být ve složce bin/debug nebo bin/release.

Windows

Stáhněte si Qt SDK s Qt >= 4.7 a gcc kompilátorem, stáhněte Lorris, otevřete Lorris.pro v QtCreatoru a stiskněte "Sestavit".

Linux

Potřebné balíčky:

- Qt SDK >= 4.7.0
- make, g++
- libudev-dev
- libsdl-dev (verze asi >= 1.2?)

apt-get pro Debian Testing by vypadalo takto: 

    sudo apt-get install qt4-dev-tools libqt4-dev qt4-qmake g++ make libsdl1.2-dev libudev-dev

Lorris také potřebuje libqwt-dev >= 6.0.0, ale od 664209866876d439e4e9b02bf156e42c62b04c76 si ji dokáže zkompilovat sám.

Teď můžete zkompilovat Lorris v QtCreatoru nebo následujícími příkazy: 

    qmake Lorris.pro
    make
    sudo make install

Lorris binárka bude instalovaná jako /usr/bin/Lorris a překlady jako /usr/share/lorris/Lorris.*.qm.
